# Project Statement.

## Overview

Spendwise. Smart Budget Tracker is a web-based personal finance application that helps users track expenses manage budgets and build savings habits in a simple way.

Spendwise is designed to be easy to use and understand.

The project focuses on solving a problem that students and individuals face. They do not know where their money is going and they cannot control their daily spending.

---

## Problem Statement

Many people have trouble with:

* Keeping track of where their money goes

* Staying within a budget

* Building savings habits

* Understanding their spending patterns

methods like writing things down or using spreadsheets are often:

* Time-consuming

* Hard to keep up with

* Not easy to understand

This leads to spending much money, poor financial planning and stress.

People need a way to manage their money.

---

## Proposed Solution

Spendwise provides a platform that allows users to:

* Set a budget for the week or month

* Easily log expenses

* Categorize spending, like food or travel

* Set limits for each category

* Create and track savings goals

* Get financial tips

Spendwise makes managing money easier by putting tracking, planning and guidance in one place.

This helps users take control of their finances.

---

## Key Functionalities

### User Authentication

* Secure login and signup using Firebase

* Guest mode for access without creating an account

### Budget Dashboard

* Shows total budget, spending, remaining balance and savings

* Visual indicators to help users understand their progress

### Budget Planner

* Allows users to set limits for each spending category

* Enables users to define savings targets

### Expense Management

* Add, view and delete expenses

* Track expenses by category and date

### Savings Goals

* Create goals

* Track progress toward targets

### Smart Tips

* Provides suggestions based on user behavior

* Encourages financial habits

---

## Technology Used

* Frontend: HTML, CSS, JavaScript

* Authentication: Firebase Authentication

* Storage: Browser LocalStorage

* Design: Custom UI with layout
