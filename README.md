# Spendwise: Smart Budget Tracker
Spendwise is a budget tracking web application that helps users manage expenses, plan budgets and build savings habits all in one place.
Spendwise is designed to be a simple and easy to use application.
The Spendwise application has a premium Deep Navy and Gold UI that offers an intuitive experience for personal finance management.
Spendwise is a tool for anyone who wants to take control of their finances.
 Features
# Authentication

•	Spendwise has a Firebase-based Sign In. Sign Up system

•	Spendwise also has a guest mode that does not require a login

•	Spendwise stores user-specific data in the browsers storage

# Dashboard

•	The Spendwise dashboard provides an overview of the users Total Budget

•	The Spendwise dashboard also shows the Total Spent and the Remaining Balance

•	The Spendwise dashboard displays the Total Savings

•	The Spendwise dashboard has a visual budget utilisation progress bar

•	The Spendwise dashboard shows a category- spending breakdown

# Budget Planner

•	The Spendwise Budget Planner allows users to set category- spending limits

•	The Spendwise Budget Planner helps users define their savings targets

•	The Spendwise Budget Planner tracks progress in time

•	The Spendwise Budget Planner sends alerts when spending limits are exceeded

# Expense Tracker

•	The Spendwise Expense Tracker allows users to add, view and delete expenses

•	The Spendwise Expense Tracker categorizes transactions

•	The Spendwise Expense Tracker logs transactions by date

# Savings Goals

•	The Spendwise Savings Goals feature allows users to create savings goals

•	The Spendwise Savings Goals feature tracks progress toward each goal

•	The Spendwise Savings Goals feature allows users to add funds incrementally

# Smart Tips Engine

•	The Spendwise Smart Tips Engine provides financial tips based on spending habits

•	The Spendwise Smart Tips Engine provides financial tips based on budget usage

•	The Spendwise Smart Tips Engine provides financial tips based on category trends

•	The Spendwise Smart Tips Engine includes practices like the 50/30/20 rule

•	The Spendwise Smart Tips Engine includes practices like the 24-hour rule

•	The Spendwise Smart Tips Engine includes practices like the weekly review habit



# Tech Stack

•	The frontend of the Spendwise application is built with HTML, CSS and JavaScript

•	The Spendwise application uses Firebase Authentication for user authentication

•	The Spendwise application uses LocalStorage for storing user data

•	The Spendwise application has a custom CSS design with a Deep Navy and Gold theme
